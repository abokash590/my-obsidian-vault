**টপিক:** কততম প্রাইম ($n$-th prime) কত হতে পারে এবং সিভ (Sieve)-এর লিমিট নির্ধারণ।

**উদ্দেশ্য:** সিভ চালানোর সময় যেন মেমোরি লিমিট বা টাইম লিমিট (TLE) ক্রস না করে, তার জন্য সঠিক রেঞ্জ বের করা।

---

### ১. মূল সূত্র: প্রাইম নম্বর থিওরেম (PNT)

গাণিতিকভাবে $n$-তম প্রাইম নম্বর ($P_n$) আনুমানিক কত হবে তা বের করার সূত্র:

$$P_n \approx n \ln n$$
কিন্তু কম্পিটিটিভ প্রোগ্রামিংয়ে আমাদের একটি **Upper Bound** বা নিরাপদ সীমা দরকার হয় যাতে কোনো প্রাইম বাদ না পড়ে। $n \ge 6$ এর জন্য একটি প্রমাণিত নিরাপদ সূত্র হলো:

$$P_n < n(\ln n + \ln(\ln n))$$

### ৩. C++ এস্টিমেশন ফাংশন

ডাইনামিকভাবে লিমিট বের করার জন্য এই ফাংশনটি তোমার টেম্পলেটে রাখতে পারো:

C++

```
/**
 * n-তম প্রাইমের জন্য আপার বাউন্ড এস্টিমেট করে।
 * সিভ অ্যারের সাইজ কত হবে তা নির্ধারণে এটি দরকারি।
 */
long long estimate_sieve_limit(int n) {
    if (n < 6) return 15; // ছোট মানের জন্য ফিক্সড
    double ln_n = log(n);
    double ln_ln_n = log(ln_n);
    // ১০% এক্সট্রা সেফটি মার্জিনসহ রিটার্ন
    return (long long)(n * (ln_n + ln_ln_n) * 1.1); 
}
```

### ৪. ইমপ্লিমেন্টেশন স্ট্র্যাটেজি (Optimized Sieve)

যেহেতু তোমার প্রায় ৪ লক্ষ প্রাইম লাগবে, তাই **৭০ লক্ষ (7,000,000)** লিমিট নেওয়া সবচেয়ে নিরাপদ।

C++

```
const int MAX_LIMIT = 7000000; 
bitset<MAX_LIMIT> is_prime;
vector<int> primes;

void sieve() {
    is_prime.set();
    is_prime[0] = is_prime[1] = 0;
    for (int p = 2; p * p < MAX_LIMIT; p++) {
        if (is_prime[p]) {
            for (int i = p * p; i < MAX_LIMIT; i += p)
                is_prime[i] = 0;
        }
    }
    for (int p = 2; p < MAX_LIMIT; p++) {
        if (is_prime[p]) primes.push_back(p);
    }
}
```

### ৫. মেমোরি এবং টাইম টিপস (Pro-Tips)

- **Standard Sieve ($10^7$):** এটি রান করতে মাত্র ~০.১ সেকেন্ড লাগে এবং `bitset` ব্যবহার করলে মাত্র ১.২ এমবি মেমোরি নেয়। প্রায় সব সিভ প্রবলেমে এটি নিরাপদ।
    
- **Segmented Sieve ($10^9$):** শুধুমাত্র তখনই ব্যবহার করো যখন $N > 10^8$ এবং মেমোরি লিমিট খুব কম (যেমন ৬৪ এমবি) থাকে।
    
- **Pre-calculating:** যদি $n$ এর মান ফিক্সড থাকে, তবে সব সময় সিভ-টি `main()` ফাংশনের বাইরে একবার ক্যালকুলেট করে রাখবে (Test Case লুপের ভেতর নয়)।
