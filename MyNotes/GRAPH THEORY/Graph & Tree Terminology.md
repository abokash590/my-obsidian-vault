## 1. Basic Graph Terminology

গ্রাফ হলো কতগুলো **Nodes (Vertices)** এবং তাদের সংযোগকারী **Edges**-এর সমষ্টি।


```

| Term | Description |
| :--- | :--- |
| **Node (Vertex)** | গ্রাফের প্রতিটি পয়েন্ট বা ইউনিট। |
| **Edge** | দুটি নোডের মধ্যকার সংযোগকারী লাইন। |
| **Degree** | একটি নোডের সাথে যতগুলো এডজ (Edge) যুক্ত থাকে। |
| **Adjacency** | যদি দুটি নোডের মধ্যে সরাসরি একটি এডজ থাকে, তবে তারা Adjacent। |
| **Path** | এক নোড থেকে অন্য নোডে যাওয়ার জন্য এডজ-এর সিকোয়েন্স। |
| **Cycle** | এমন একটি পাথ যা যে নোড থেকে শুরু হয়েছে সেখানেই শেষ হয়। |

```

### **Types of Graph:**

1. **Directed (Digraph):** এডজগুলোর নির্দিষ্ট দিক থাকে ($u \rightarrow v$)।
2. **Undirected:** এডজগুলোর কোনো দিক থাকে না ($u \leftrightarrow v$)।
3. **Weighted:** প্রতিটি এডজ-এর একটি নির্দিষ্ট মান (Weight/Cost) থাকে।
4. **Disconnected:** গ্রাফের সব নোড একে অপরের সাথে যুক্ত থাকে না।

## 2. Tree Terminology

একটি **Tree** হলো এমন একটি গ্রাফ যেখানে কোনো **Cycle** নেই এবং সব নোড কানেক্টেড।


```

| Term | Description |
| :--- | :--- |
| **Root** | ট্রির একদম উপরের নোড যেখান থেকে সব শুরু হয়। |
| **Leaf Node** | যে নোডের কোনো চাইল্ড (Child) নেই। |
| **Parent/Child** | উপরের নোডটি Parent এবং নিচেরটি Child। |
| **Siblings** | একই প্যারেন্টের আন্ডারে থাকা চাইল্ডরা একে অপরের সিবলিং। |
| **Height** | রুট থেকে লিফ নোড পর্যন্ত দীর্ঘতম পাথের এডজ সংখ্যা। |
| **Depth** | রুট থেকে নির্দিষ্ট একটি নোড পর্যন্ত এডজ সংখ্যা। |
| **Subtree** | ট্রির যেকোনো একটি নোড এবং তার নিচের সব চাইল্ড মিলে একটি সাব-ট্রি। |

```

## 3. Important Concepts (Special Graphs)

১. **DAG (Directed Acyclic Graph):** এমন একটি ডিরেক্টেড গ্রাফ যেখানে কোনো সাইকেল নেই। এটি টপোলজিক্যাল সর্টিংয়ের জন্য ব্যবহৃত হয়। ২. **Bipartite Graph:** গ্রাফের নোডগুলোকে দুটি সেটে ভাগ করা যায় যাতে কোনো সেটের নিজের নোডগুলোর মধ্যে কোনো এডজ না থাকে। ৩. **Complete Graph:** প্রতিটি নোড অন্য প্রতিটি নোডের সাথে সরাসরি যুক্ত থাকে। ৪. **Spanning Tree:** একটি কানেক্টেড গ্রাফের এমন একটি সাব-গ্রাফ যা সব নোডকে কভার করে কিন্তু কোনো সাইকেল তৈরি করে না।

## 4. Graph Connectivity & Structure

গ্রাফের ভেতরকার সংযোগ এবং শক্তি বোঝার জন্য এই টার্মগুলো খুবই গুরুত্বপূর্ণ।

Markdown

```

| Term | Description |
| :--- | :--- |
| **Connected Component** | গ্রাফের এমন একটি সাব-গ্রাফ যেখানে যেকোনো দুটি নোডের মধ্যে পথ থাকে। |
| **Strongly Connected (SCC)** | ডিরেক্টেড গ্রাফের ক্ষেত্রে, যদি $u \rightarrow v$ এবং $v \rightarrow u$ উভয় পথই থাকে। |
| **Articulation Point** | এমন একটি নোড যা রিমুভ করলে গ্রাফের কানেক্টেড কম্পোনেন্ট সংখ্যা বেড়ে যায়। |
| **Bridge** | এমন একটি এডজ যা রিমুভ করলে গ্রাফটি ডিসকানেক্টেড হয়ে যায়। |
| **Biconnected Graph** | যে গ্রাফে কোনো আর্টিকুলেশন পয়েন্ট নেই। |
| **Clique** | গ্রাফের একটি সাব-সেট যেখানে প্রতিটি নোড অন্য সবার সাথে সরাসরি যুক্ত। |

```

## 5. Advanced Tree Concepts

ট্রি-র প্রোপার্টিজ এবং স্পেশাল টাইপগুলো নিচে দেওয়া হলো:

Markdown

```

| Term | Description |
| :--- | :--- |
| **Diameter of Tree** | ট্রির যেকোনো দুটি নোডের মধ্যে দীর্ঘতম দূরত্ব। |
| **Subtree Size** | একটি নির্দিষ্ট নোডকে রুট ধরে তার নিচের মোট নোড সংখ্যা। |
| **Lowest Common Ancestor (LCA)** | দুটি নোডের সবচেয়ে কাছের কমন পূর্বপুরুষ বা প্যারেন্ট নোড। |
| **Binary Tree** | যে ট্রিতে প্রতিটি নোডের সর্বোচ্চ দুটি চাইল্ড থাকে। |
| **BST (Binary Search Tree)** | লেফট চাইল্ড < রুট < রাইট চাইল্ড—এই শর্ত মেনে চলা ট্রি। |
| **Balanced Tree** | যে ট্রির লেফট এবং রাইট সাব-ট্রির হাইট প্রায় সমান (যেমন: AVL, Red-Black Tree)। |

```

## 6. Graph Traversal & Path Terms

অ্যালগরিদম ইমপ্লিমেন্ট করার সময় এই শব্দগুলো বারবার আসবে:

- **In-degree:** একটি নোডের দিকে যতগুলো ডিরেক্টেড এডজ আসে।
- **Out-degree:** একটি নোড থেকে যতগুলো ডিরেক্টেড এডজ বাইরের দিকে যায়।
- **Source:** যে নোডের ইন-ডিগ্রী ০।
- **Sink:** যে নোডের আউট-ডিগ্রী ০।
- **Eulerian Path:** এমন একটি পথ যা গ্রাফের প্রতিটি **এডজ** ঠিক একবার ব্যবহার করে।
- **Hamiltonian Path:** এমন একটি পথ যা গ্রাফের প্রতিটি **নোড** ঠিক একবার ভিজিট করে।

## 7. Tree Types (Based on Structure)

১. **Full Binary Tree:** প্রতিটি নোডের হয় ০ অথবা ২ টি চাইল্ড থাকে। ২. **Complete Binary Tree:** শেষ লেভেল বাদে সব লেভেল পূর্ণ থাকে এবং শেষ লেভেলের নোডগুলো বাম দিক থেকে সাজানো থাকে। ৩. **Perfect Binary Tree:** সব ইন্টারনাল নোডের ২ টি চাইল্ড থাকে এবং সব লিফ নোড একই লেভেলে থাকে। ৪. **Degenerate (Skewed) Tree:** প্রতিটি প্যারেন্ট নোডের মাত্র একটি চাইল্ড থাকে (এটি অনেকটা লিঙ্কড লিস্টের মতো কাজ করে)।
